Mini Hit

Unity C# Action

About the Game

Mini Hit is a fast-paced action game where quick reflexes and precise timing are key. Built with Unity and C#, this game offers an engaging gameplay experience directly in your browser.

Key Features

  • Fast-paced action gameplay
  • Browser-based experience
  • Intuitive controls
  • Progressive difficulty

Development Journey

Mini Hit was developed with Unity game engine as a school project in the first place. Using the help of a youtube tutorial, it was running in less than a few hours.

Technical Highlights

  • Optimized for web deployment (current bugs will be fixed in the near future)
  • Responsive input system
  • Efficient collision detection
  • Performance-focused design
