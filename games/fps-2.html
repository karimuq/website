FPS-2

Unity C# FPS

About the Game

FPS-2 is a first-person shooter game that features dynamic environments and custom mechanics. Built with Unity and C#, this game showcases advanced gameplay elements and interactive environments.

Key Features

  • Dynamic environment interaction
  • Custom weapon mechanics
  • Advanced AI behavior
  • Multiple game modes

Development Journey

The development of FPS-2 started as an exploration of 3D game mechanics in Unity. Throughout the process, I focused on creating smooth player controls and engaging combat mechanics.

Technical Highlights

  • Implemented custom physics-based movement system
  • Created modular weapon system
  • Developed AI pathfinding and behavior trees
  • Optimized performance for smooth gameplay
