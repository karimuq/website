O-Cars

Unity C# Racing

About the Game

O-Cars is a racing game demo that focuses on realistic physics and engaging vehicle mechanics. Players can experience different cars with unique handling characteristics on various tracks.

Key Features

  • Realistic vehicle physics
  • Multiple car types
  • Dynamic race tracks
  • Time trial mode

Development Journey

O-Cars was developed to explore vehicle physics and racing game mechanics in Unity. The project focused on creating realistic car behavior while maintaining fun gameplay elements.

Technical Highlights

  • Custom vehicle physics system
  • Advanced tire friction model
  • Dynamic camera system
  • Race position tracking
